# i3installscript
#
# To Install
first install git
```
sudo pacman -S git
```
and then
```
cd ~/ && git clone https://github.com/SometimesSquishy/i3installscript.git
```
# To run
```
cd ~/i3installscript/i3installconfig && chmod +x install.sh && ./install.sh
```
